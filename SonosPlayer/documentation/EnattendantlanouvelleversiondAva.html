En attendant la nouvelle version d'Avatar

Une petite mise à jour doit être effectuée sur le Serveur et les clients A.V.A.T.A.R.


Cette action sera supprimée lorsqu'une nouvelle version d'A.V.A.T.A.R sera mise à disposition. En attendant, pour que la commande Play redirige correctement les musiques à jouer sur les devices Sonos, procédez comme suit:


  1. Télécharger le plug-in SonosPlayer depuis sa page Github
  2. Dé-zippez le fichier Avatar-Plugin-SonosPlayer
  3. Puis dans ce répertoire dé-zippé:

Sur le serveur:

  1. Déplacez-vous dans le répertoire updatePlay/Serveur
  2. Copiez le répertoire resouces dans le répertoire <A.V.A.T.A.R Serveur>
  3. Acceptez la demande de remplacement du fichier speech.js


Sur les clients Windows:

  1. Déplacez-vous dans le répertoire updatePlay/Clients
  2. Copiez le répertoire app dans le répertoire <A.V.A.T.A.R Client>
  3. Acceptez la demande de remplacement des fichiers avatar.js et listen.js
