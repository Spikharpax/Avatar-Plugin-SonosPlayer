Paramètres annexes Spotify

Pour Spotify uniquement, vous avez besoin de 2 paramètres supplémentaires.

Ces paramètres sont déjà configurés pour une utilisation en Europe et en France:



Pour les Etats-Unis, le code region est US

Pour un autre pays, modifiez la propriété available_market avec le code de votre pays.
