Un réveil ?


Il y a 2 actions que vous pouvez utiliser dans le plugin "scenariz" pour créer un scénario qui déclenche de la musique aux jours et heure de votre choix en cherchant dans votre bibliothèque musicale (hors Spotify).



Nom de l'action

Paramètre

Valeur de paramètre



wakeup_random_music

searchType

radio, artists, albums, genres, composers, tracks, playlists

searchTerm

En association avec searchType (voir ci-dessus), soit dans l'ordre:

Un nom de radio, d'artiste, un genre, un compositeur, une piste, une playliste

ou

null pour le type si vous ne voulez pas préciser de terme de recherche.


Dans tous les cas, A.V.A.T.A.R prendra au hasard un résultat parmi la liste de résultat pour le jouer.

wakeup_volume

value

un entier de 0 à 100 pour le niveau de volume



Note:

Le paramètre "_id" dans les exemples ci-dessous doit être unique.

Si vous avez des "_id" identiques, modifiez en un avec une lettre ou un chiffre différent.


Exemple 1:

Dans cet exemple, je veux un scénario qui est joué dans la pièce Sonos "Chambre" à 7:45 les jours de la semaine et qui cherche un artiste "Pat Benatar".

Mon programme "Réveil" contient une action du nom "musique de réveil".

(Analysez les paramètres searchType et searchTerm dans le scénario ci-dessous)


{"Program":"Réveil","Client":"Chambre","Plugin":"SonosPlayer","Name":"musique de reveil","Order":"1","Tempo":"1000","Speech":null,"Exec":"true","Keys":"command=wakeup_random_music~searchType=artists~searchTerm=Pat Benatar~client=Chambre","Hour":"07:45","Days":"1111100","_id":"TsE35Fv1ZcBwMwcI"}


Cette ligne est à ajouter dans le fichier <A.V.A.T.A.R>\resources\core\plugins\scenariz\db\Scenariz.db

Vous pouvez ensuite le modifier vocalement par la commande "Gestion des programmes" du plugin scenariz (Pour plus d'information, voir le plugin scenariz)
