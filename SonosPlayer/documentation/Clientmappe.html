Client mappé

Si vous désirez avoir les dialogues d'A.V.A.T.A.R sur un appareils Sonos qui se trouve dans une pièce mappée, vous devez configurer A.V.A.T.A.R.


Note importante:

Pour qu'A.V.A.T.A.R puisse reconnaître la pièce dans laquelle vous vous trouvez, il est obligatoire d'avoir des capteurs de présences dans la pièce "réelle" et les pièces mappées. Configurez votre box domotique pour qu'elle envoie à A.V.A.T.A.R la pièce courante. Consultez la page d'information du plug-in "Les standards" au chapitre "Définir la pièce courante" pour plus d'information.



  1. Dans l'interface d'A.V.A.T.A.R serveur
  2. Sur le Node serveur, cliquez sur "Menu Plugins-> Plugin Studio"
  3. Cliquez ensuite sur le plug-in "Sonos" puis sur l'onglet "Propriétés"
  4. En dessous de mapped_client_speak, cliquez sur le petit symbole à gauche de (empty object)
  5. Dans le menu "Append", cliquez sur "Auto"


       


  1. Entrez le nom du client mappé

       


  1. Pour ajouter une autre pièce, cliquez sur le petit symbole à gauche d'une pièce
  2. Dans le menu "Append", cliquez sur "Auto"
  3. Sauvegarder vos modification (clique droit) -> Sauvegarder


Dans l'exemple ci-dessus, le client mappé "Cuisine" qui n'est pas un client réel aura ses dialogues d'A.V.A.T.A.R sur l'appareil Sonos "Cuisine" mais vous utiliserez le micro du client réel avec lequel ce client est mappé.
